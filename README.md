## Project Overview

This project simulates a real world analytics lifecycle:
1. Data ingestion from raw sources
2. Data cleaning and validation
3. Data transformation and modelling
4. Exploratory data analysis
5. Insight generation
6. Dashboard creation for business users
The focus is on clarity, structure, and production style workflow rather than just notebooks.
